6 external data sources — yfinance, FRED, SEC EDGAR, Quiver Quant, Google Trends, CME FedWatch
▸
3 LLM providers — Anthropic (Claude Sonnet 4.6), xAI (Grok 4.1), Groq (llama-3.3-70b)

Prompt Grounding Architecture signal_block.py
▸
build_macro_block() — injected at TOP of every AI prompt. Contains: regime quadrant, macro score, 26 z-scores, Fed rate, tactical score, rate path, upcoming FOMC/CPI/NFP dates with urgency flags.
▸
build_ticker_block(ticker) — injected into ticker-specific prompts (valuation, filing). Contains: price, RSI, options P/C, insider bias, institutional %, earnings date with urgency flag.
▸
get_signal_fingerprint() — MD5 hash of macro z-scores (1dp). Busts cache when macro regime changes significantly.
▸
get_ticker_fingerprint(ticker) — MD5 hash of ticker signals + macro fingerprint. Cache key for _valuation_cache.
